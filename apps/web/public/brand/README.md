# Brand assets

- `logo-black.png` / `logo-white.png` — legacy marks (optional)
- Favicon and OG image are generated by `scripts/generate-brand-assets.mjs` and `scripts/generate-favicons.mjs`
- In-app wordmark: `components/brand/hex-wordmark.tsx` (no Lottie)

Regenerate after visual changes:

```bash
pnpm --filter @steprs/web generate:brand-assets
pnpm --filter @steprs/web favicons
```
