# steprs-mcp (private)

The MCP server for STEP edit sessions (`open_session`, `plan_edit`, `apply_edits`, `export_step`) is **not included in the public steprs-dev repository**.

It ships with the proprietary geometry editor in a private repository. The public workbench returns **503** on all `/api/editor/*` routes.

See [docs/PRIVATE_EDITOR.md](../../docs/PRIVATE_EDITOR.md).
